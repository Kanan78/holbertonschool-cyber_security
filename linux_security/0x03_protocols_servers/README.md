### üì° Core Network Protocols

| Protocol | Purpose                                  |
| -------- | ---------------------------------------- |
| TCP/IP   | Core communication suite of the Internet |
| UDP      | Fast, connectionless data transmission   |
| ICMP     | Error reporting & diagnostics            |
| DNS      | Domain name resolution                   |
