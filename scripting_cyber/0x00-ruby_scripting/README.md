# Ruby in Cybersecurity

## Description

This project introduces Ruby programming in the context of cybersecurity.
Ruby is the core language of the Metasploit framework, the most widely used
penetration testing platform in the industry. By working through these tasks,
learners build practical tools such as port scanners, password testers, and
reverse shells while developing a deep understanding of both attacker
techniques and defensive strategies.

Each script in this module translates code into real-world security insight.
This practical approach fosters a proactive security mindset and builds the
technical foundation required for professional certifications such as OSCP
and PenTest+.

---

## Why Ruby in Cybersecurity?

- **Metasploit Framework**: Ruby is the core language of Metasploit,
  essential for writing custom modules and payloads.
- **Automation and Scripting**: Automate repetitive security tasks and
  integrate various tools and systems.
- **Custom Exploits and Payloads**: Develop custom exploits for new
  vulnerabilities and customize payloads to evade detection.
- **Security Research**: Write proof-of-concept code for vulnerabilities
  and contribute to open-source security tools.
- **Flexibility and Readability**: Clean syntax supports rapid development
  and prototyping.
- **Versatility**: Applicable across web application security, network
  security, and forensic analysis.

## Requirements

- Allowed editors: `vi`, `vim`, `emacs`
- All scripts are tested on **Kali Linux**
- Substitute the IP range using `$1` (without quotes)
- All files must end with a new line
- First line of every file must be exactly: `#!/usr/bin/env ruby`
- Code must follow the **Betty style** (checked with `betty-style.pl`
  and `betty-doc.pl`)
- All files must be **executable**
- Backticks, `&&`, `||`, and `;` are **not allowed**

---

## Learning Objectives

By the end of this project, you should be able to explain without Google:

- Why Ruby is used in cybersecurity
- How the Metasploit framework is structured
- How to write and customize exploits and payloads
- How to automate security tasks with Ruby scripts
- How to build tools for port scanning, banner grabbing, and
  vulnerability detection
- How to perform SSH brute force, SQL injection detection, and
  XSS detection
- How to simulate a man-in-the-middle attack and packet sniffing
- How to analyze malware, create reverse shells, and build keyloggers
- How to interact with databases, REST APIs, and web services in Ruby
